# Lab: License And Group Review

## Scenario

A user cannot access a Microsoft 365 app. Check whether the user has the required cloud group or license assignment.

## Tasks

1. Confirm which app is affected.
2. Inspect the user's cloud account.
3. Review assigned licenses.
4. Review group membership.
5. Add the user to an approved licensing group if your lab supports group-based licensing.
6. Ask the user to retest.
7. Record the result in the ticket.
