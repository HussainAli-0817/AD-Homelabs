# Lab: MFA And Sign-In Support

## Scenario

A user changed phones and cannot complete MFA. Review the user's authentication methods, document the support action, and provide a clean closure note.

## Tasks

1. Verify the user's identity.
2. Confirm the affected account.
3. Review authentication methods.
4. Follow your tenant's approved process for updating or re-registering MFA.
5. Ask the user to test sign-in.
6. Document the final state.

## Notes

MFA reset permissions are sensitive. In real environments, follow the organization's access model and escalation path.
