# Entra ID Labs

These labs build cloud identity skills for Microsoft 365 and help desk support.

## Prerequisites

- Microsoft Graph PowerShell SDK installed.
- Training or developer tenant access.
- Permissions approved by the tenant owner.

## Labs

- `mfa-conditional-access/`: review MFA registration and user sign-in support.
- `license-group-management/`: inspect cloud user licenses and groups.

## Recommended Module

Use Microsoft Graph PowerShell for modern Entra ID automation. Avoid building new workflows around the retired AzureAD and MSOnline modules.
