# Lab: Password Reset And Unlock

## Scenario

A user cannot sign in and reports repeated password prompts. Verify the account status, unlock the account if needed, reset the password, and require a password change at next sign-in.

## Tasks

1. Verify the user's identity.
2. Inspect account state.
3. Unlock the account if locked.
4. Reset the password.
5. Require password change at next sign-in.
6. Ask the user to test sign-in.
7. Document the result.
