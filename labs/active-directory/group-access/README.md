# Lab: Group Access Request

## Scenario

A manager approved access to a file share. Add the user to the correct security group and document the approval.

## Tasks

1. Confirm the request includes manager or data owner approval.
2. Verify the target group exists.
3. Verify the user exists.
4. Add the user to the group.
5. Confirm membership.
6. Tell the user to sign out and back in if required.
7. Close the ticket with the group name and approval reference.

## Ticket Tip

Never grant access based only on a casual chat message. Record who approved it and where that approval is stored.
