# Help Desk Role Map

Use this map to connect each lab to real help desk responsibilities.

| Responsibility | AD Skill | Entra ID Skill | Ticket Evidence |
| --- | --- | --- | --- |
| Password reset | Reset AD password, unlock account | Verify sign-in status | User verified, password reset, temporary password handled securely |
| Account lockout | Inspect `LockedOut`, bad password count, event logs | Check risky sign-ins where available | Lockout cause, device or app identified, user advised |
| New hire onboarding | Create user, place in OU, assign groups | Assign cloud groups and licenses | Request approval, groups added, start date noted |
| Access request | Modify security group membership | Modify cloud group membership | Approval, group name, access owner, completion note |
| Offboarding | Disable account, remove groups, move OU | Block sign-in, remove licenses | HR request, effective date, actions completed |
| MFA support | N/A for AD DS | Review authentication methods and registration | Identity verified, MFA method updated or user guided |
| License issue | N/A for AD DS | Inspect assigned licenses and service plans | License state, change made, app retest |

## What Hiring Managers Want To See

- You can follow a ticket from intake to closure.
- You verify the user's identity before sensitive changes.
- You ask for approval before granting access.
- You write clean internal notes.
- You know the difference between AD DS and Entra ID.
- You can explain what you changed and how you validated it.
